**Phantom Launcher** is a high-fidelity, sci-fi themed game launcher designed to organize your game library with a premium, immersive aesthetic. 

> [!IMPORTANT]
> **Platform Support**: This application is currently **Windows-only**.

### Features
*   **Deep Immersion UI**: Cyberpunk/Sci-fi aesthetic with CRT effects and neon accents.
*   **Universal Library**: Manage games from Steam, Xbox, and manual entries.
*   **Internationalization**: Fully localized in English and Spanish.
